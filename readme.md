## 使用方法

* 点击右边栏的 `Releases` 后，下载 `app.asar` 或者 下载 `app.asar.zip` 解压

* 然后按照 `Releases` 页面的指引，将 `app.asar` 放在指定目录下

* 打开后，选择 `帮助` - `输入许可证密钥`，随便输入或者直接点击OK即可
